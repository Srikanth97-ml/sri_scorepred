# IPL Score Prediction using Machine Learning

This project predicts the final score of an IPL match using machine learning.

## Technologies
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Score Prediction

The trained model achieved an **R² score of 0.87 on the test dataset**, showing good predictive performance for estimating IPL first-innings scores.
